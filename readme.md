# FengShui

Fengshui est un site internet qui présente le fengsui et propose des consultations.

## Environnement de développement

### Pré-requis

 * PHP 8.1
 * Composer
 * Symfony CLI
 * Docker
 * Docker-compose


### lancer l'environnement de développement

```bash
docker-compose up -d
stmfony serve -d 

```
